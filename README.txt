PBH SIMULATION COMPLETE ANALYSIS PACKAGE
==================================================

Este paquete contiene todos los resultados de la simulación de
Agujeros Negros Primordiales usando el marco de Tiempo Aplicado.

ARCHIVOS INCLUIDOS:
- figures/: Gráficos profesionales de alta resolución
- tables/: Datos numéricos completos en CSV
- reports/: Reportes ejecutivos y técnicos
- models/: Documentación de modelos matemáticos
- analysis/: Análisis de sensibilidad y convergencia
- Simulation_Configuration.json: Configuración completa

REFERENCIA: Percudani, M.A. - Evolution of Primordial Black Holes
